# CASUR Maps V20.3 · Header oficial + control de capas libre

Esta versión aplica dos ajustes quirúrgicos sobre V20.2:

1. Agrega el logo oficial CASUR al header del panel lateral.
2. Retira el badge flotante “CASUR Maps” del mapa para que no tape el control de selección de capas/base map de Leaflet.

## Header visible

- Logo CASUR · Compañía Azucarera del Sur, S.A.
- Departamento de Negocios de Caña
- CASUR Maps

## Corrección de capas

La esquina superior derecha del mapa queda libre para el control de capas. El badge flotante se oculta y se mantiene el nombre CASUR Maps dentro del header oficial.

## Instalación

Suba todo el contenido interno de esta carpeta a la raíz del repositorio GitHub Pages. Luego limpie caché o reinstale la PWA si el celular conserva la versión anterior.
