# CASUR Maps V21.5 · Compartir Hallazgo PRO

Esta versión amplía la tarjeta de hallazgo de V21.4.

## Nuevo
- Técnico / responsable.
- Categoría del hallazgo.
- Severidad.
- Foto opcional.
- La tarjeta PNG incluye los datos del lote, métricas principales, comentario, coordenadas, fecha/hora y evidencia fotográfica si fue adjuntada.
- Se mantiene compartir/WhatsApp, descargar imagen y copiar texto.

## Uso
1. Seleccione un lote.
2. Toque **Compartir hallazgo**.
3. Complete técnico, categoría, severidad, comentario y foto opcional.
4. Genere, comparta, descargue o copie el texto.

## Instalación
Suba todo el contenido interno de esta carpeta a la raíz de GitHub Pages. Luego limpie caché o reinstale la PWA.
