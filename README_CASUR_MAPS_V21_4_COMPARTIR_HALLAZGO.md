# CASUR Maps V21.4 · Compartir Hallazgo

Esta versión agrega un módulo rápido para generar una **tarjeta de hallazgo** a partir del lote seleccionado y compartirla.

## Incluye
- Botón **Compartir hallazgo** en la tarjeta del lote.
- Comentario rápido / hallazgo.
- Generación de tarjeta visual en PNG.
- Compartir vía hoja nativa del teléfono (ideal para WhatsApp cuando el navegador lo soporta).
- Descarga de imagen PNG.
- Copia de texto como respaldo.

## Instalación
Suba el contenido interno de esta carpeta a la raíz del repositorio GitHub Pages y limpie caché o reinstale la PWA.
