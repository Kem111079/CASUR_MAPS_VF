- Se redujo el espacio de las tarjetas/métricas.
- Las métricas ahora se muestran en una fila compacta.
- Se amplió el área del comentario.
- Se amplió el área de la foto.
- La foto se muestra completa, sin recorte, usando modo `contain`.
- Se conserva compartir/WhatsApp, descargar imagen y copiar texto.

## Recomendación de prueba
1. Seleccionar un lote.
2. Abrir Compartir hallazgo.
3. Adjuntar foto vertical y horizontal.
4. Generar tarjeta.
5. Verificar que la foto se vea completa y que el comentario tenga suficiente espacio.
