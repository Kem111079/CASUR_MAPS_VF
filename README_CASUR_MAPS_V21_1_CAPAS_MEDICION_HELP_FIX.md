# CASUR Maps V21.1 · Capas y medición PRO

Cambios principales:

1. El selector de capas de Leaflet queda siempre accesible desde el inicio, sin depender del estilo satélite.
2. La herramienta de medición ahora muestra barra flotante al medir longitud o área.
3. La medición permite revisar todas las unidades: distancia en m y km; área en m², ha y mz.
4. Se agregó selector de unidad destacada en modo Auto/m/km y Auto/m²/ha/mz.
5. El panel lateral incorpora módulos colapsables con breve ayuda de uso para reducir saturación visual.
6. Mantiene el header institucional: Departamento de Negocios de Caña / CASUR Maps / Mapa inteligente de lotes cañeros.

## Instalación
Suba el contenido interno de esta carpeta a la raíz de GitHub Pages y limpie caché/reinstale la PWA para forzar la nueva versión.
