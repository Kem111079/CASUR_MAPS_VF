# CASUR Maps V19.2 · Icon Update C + Pin

Esta versión actualiza el branding visual de la PWA usando el icono aprobado **Propuesta 3 · C + Pin**.

## Cambios
- Reemplazo del icono principal por el nuevo diseño C + Pin.
- Nuevos tamaños en `icons/`: 72, 96, 128, 144, 152, 180, 192, 384 y 512 px.
- Nuevo `maskable-icon-512.png`.
- Nuevos favicons (`favicon-16`, `favicon-32`, `favicon-48`, `favicon.png`).
- `manifest.json` actualizado.
- `index.html` actualizado con apple touch icon y favicons.
- `service-worker.js` actualizado a `casur-maps-pwa-v19-2-icon-update-c-pin`.

## Instalación
Sube todo el contenido interno de esta carpeta a la raíz del repositorio GitHub Pages, reemplazando los archivos existentes. Luego desinstala la PWA anterior o limpia caché y vuelve a instalar para ver el nuevo icono.
